# Project Features

## Core Tech Stack
- **Frontend Framework**: React with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS with custom theme configuration
- **Animations**: Framer Motion, React Spring
- **3D Graphics**: React Three Fiber (Three.js)
- **Backend/Auth**: Supabase
- **State Management/Data Fetching**: React Query
- **Routing**: React Router DOM

## User Roles & Authentication
- **Authentication System**:
  - Sign Up & Login (Email/Password)
  - Role-based Access Control (Candidate, Recruiter, Admin)
  - Protected Routes
  - Persistent Session Management

## Landing Page
- **Hero Section**: 3D interactive elements and engaging visuals.
- **How It Works**: Step-by-step guide for users.
- **Features Section**: Highlight of key platform capabilities.
- **Jobs Preview**: Teaser of available job listings.
- **Opportunity Carousel**: Rotating showcase of opportunities.
- **Recruiter Marquee**: Display of top hiring companies.

## Candidate Features
- **Dashboard**: Personalized overview of applications and status.
- **Job Board**:
  - Full list of available jobs.
  - Advanced filtering and search capabilities.
- **Job Details**: Comprehensive view of job descriptions, requirements, and salary.
- **Application System**:
  - Direct job application flow.
  - File upload for resumes/CVs.
  - Application Timeline tracking.
- **Profile Management**: Editable candidate profile.
- **ATS Checker**:
  - Resume analysis tool.
  - PDF parsing and keyword matching.
  - Score and feedback generation.

## Recruiter Features
- **Dashboard**: Overview of posted jobs and recent activities.
- **Job Management**:
  - Create and post new job listings.
  - Manage existing listings.
- **Applicant Management**:
  - View list of applicants for specific jobs.
  - Review applicant details.

## Admin Features
- **Admin Dashboard**: High-level platform metrics and overview.
- **User Management**:
  - View and manage all registered users (Candidates, Recruiters).
- **Job Management**:
  - Oversight of all job postings on the platform.
  - Ability to remove or edit listings.
- **Application Analytics**: Visual data on application trends and volume.
- **Recruiter Profiles**: Detailed view and management of recruiter profiles.
- **Global Applicant View**: Access to view applicants across all jobs.

## Legal & Compliance
- **Terms of Service**: Detailed terms of use.
- **Privacy Policy**: Data handling and privacy information.

## UI/UX Features
- **Responsive Design**: Mobile-first approach.
- **Dark/Light Mode**: (Implied by ThemeProvider/Context).
- **Custom Cursor**: Interactive cursor spotlight effect.
- **Page Transitions**: Smooth transitions between routes.
- **Interactive Components**: Modals, Toasts, and interactive cards.
